# Session 15 & 16 (Combined) – Final Discussion

This session focuses on **wrapping up the mentorship**, connecting what you’ve learned, and understanding how to move forward using your existing project.

## 1. Remote Work Basics & Communication

- What remote work actually looks like in real life
- Importance of clear and timely communication
- Working with people across time zones
- Basic meeting and messaging etiquette (Google Meet, Slack, email)
- Common beginner mistakes in remote environments

## 2. Using the Project as a Portfolio

- How the built project represents your skills
- What parts of the project matter most for beginners
- Explaining your project in simple terms:
  - What problem it solves
  - What technologies were used
  - What you personally learned
- Why perfection is not required for a beginner portfolio

## 3. LinkedIn Profile Basics

- Writing a simple and honest headline
- What to include in the “About” section
- Adding your project link in the Featured section
- Keeping the profile realistic for an entry-level learner

## 4. GitHub Profile Basics

- Making repositories public
- What a basic README should contain:
  - Project overview
  - Tech stack
  - Live project link
- Why clean code is less important than real work at this stage

## 5. What to Do After the Program

- Improving the same project gradually
- Building one more small project
- Using the portfolio for internships, junior roles, or freelancing
- Continuing self-learning with consistency

## Notes

- Project submission is **not mandatory**
- No presentation or peer review is required
- Students are encouraged (but not forced) to finish and improve their projects independently

**This marks the final session of the mentorship program.**
